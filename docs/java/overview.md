# Java Практики

<img src="/img/java-icon.svg" alt="Java icon" width="56" />

Java-трек состоит из 24 практических задач и последовательно проводит от базового синтаксиса к архитектурным паттернам и модульному проектированию.

## Основные темы

- классы, объекты, инкапсуляция, наследование;
- интерфейсы, коллекции, обобщения и очереди;
- исключения и файловая система;
- GUI и паттерны проектирования;
- итоговый проект на основе системы заказов.

## Формат модулей

Каждый модуль содержит:

- теоретический блок;
- кодовый пример по теме;
- практический разбор;
- важные замечания и граничные случаи;
- итоговый вывод.

## Исходники

Все задачи расположены в `pr_Java/`.
