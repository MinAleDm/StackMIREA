## Теория

Регулярные выражения позволяют описывать текстовые шаблоны и быстро извлекать данные. Комбинаторы (`|`, группы, квантификаторы) помогают строить гибкие правила.

## Код

```python title="practice_6_regex.py"
import re

pattern = re.compile(r"^(?:\+7|8)\d{10}$")

phones = ["+79991234567", "89991234567", "12345"]
for phone in phones:
    print(phone, "OK" if pattern.match(phone) else "INVALID")
```

## Разбор

Шаблон принимает два формата телефонного номера: `+7` или `8`, затем ровно 10 цифр. Группа `(?:...)` используется без сохранения в отдельную подмаску.

Исходник практики: `pr_Python/pr_Python/pract6.ipynb`.

## Важные моменты
